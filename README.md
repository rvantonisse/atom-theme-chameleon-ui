# CHAMELEON-UI

Adaptable Atom UI theme, cloning your favourite syntax color schemes.

## Colors used from syntax theme

All we need is:

- \@syntax-text-color
- \@syntax-background-color
- \@syntax-color-info;
- \@syntax-color-success;
- \@syntax-color-warning;
- \@syntax-color-error;
